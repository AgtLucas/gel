A GelUI Sampler: The Link Box Component

When to Use This Component

The GelUI Link Box Component is based on the HTML Anchor <A> element, but has visual enhancements to give it a more button-like appearance: having a rectangular border and an optional icon. Semantically however, an Anchor implies navigation, whilst a button implies an action. So in spite of its standout appearance, this component is still semantically an Anchor and should only be used to navigate the browser somewhere.

Example GelUI Link Box

Try It

Example source code

React.createElement(
  GelLinkBox,
  {
    theme: 'primary', // default
    label: 'Apply now',
    href: '#event-application',
    icon: 'next', // optional
    attributes: { // optional
      'data-trackid': '8A52-E183-5D15',
      'aria-label': 'Apply now to attend this event.'
    },
    onClick: function( e ) { alert( 'You clicked "' + this.props.label + '".' ); }
  }
);

Example static output

<a data-trackid="8A52-E183-5D15" aria-label="Apply now to attend this event." href="#event-application" class="gel-button gel-button__link-button gel-button--primary">
  <span class="gel-button__content">Apply now</span>
  <span class="gel-button__icon_wrapper">
    <span class="gel-button__icon--next">
      <svg class="gel-icon gel-icon--text"><use xlink:href="assets/icons.svg#gel-icon-next" style: "fill:undefined;" /></svg>
    </span>
  </span>
</a>

More GelUI Link Box Examples

Configure the icon

React.createElement(
  GelLinkBox, {
  label: 'Visit related site',
  icon: 'external-link',
  href: '#visit-related'
});

Configure the theme

React.createElement(
  GelLinkBox, {
  theme: 'danger',             
  label: 'Exit',
  icon: 'no',
  href: '#back-to-start'
});

Use the light theme

React.createElement(
  GelLinkBox, {
  theme: 'light',
  label: 'Download forms (MicroSoft Word Doc - 330kB)',
  icon: 'download',
  href: '#download-forms'
});

Add your own style

React.createElement(
  GelLinkBox, {
  theme: false,
  label: 'Take the survey',
  href: '#take-survey',
  className: 'mylinkbox'
});
